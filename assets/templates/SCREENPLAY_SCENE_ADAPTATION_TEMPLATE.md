# Screenplay Scene Adaptation Template

For each scene, use this format:

SCENE NUMBER:
[Number in source order]

SOURCE RANGE:
[Source file and line range]

LOCATION:
[Use source location only. If missing, write unknown.]

TIME:
[Use source time only. If missing, write unknown.]

CHARACTERS PRESENT:

* [Only characters physically present or clearly active in this scene.]

SITUATIONAL CUES PRESENT:
List concrete source cues that should be preserved in the scene:

* exhaustion, hesitation, silence
* weather, lighting, sound
* objects handled
* room condition
* clothing
* posture, gesture, physical behavior
* distance between characters
* entrances/exits
* physical routines or business

Do not interpret these cues emotionally. Preserve them as visible or audible screen action.

SOURCE DIALOGUE INSIDE THIS SCENE:
Copy only the exact dialogue that belongs to this scene.
Only include spoken lines, inner quoted thoughts, or clearly attributed speech. Do not include narration, setting description, crowd description, or action prose as dialogue.

Rules:

* Do not move dialogue from another scene.
* Do not summarize dialogue.
* Do not paraphrase dialogue.
* Do not invent dialogue.
* Preserve exact wording.
* If speaker is unclear, write SPEAKER UNKNOWN.
* If no exact dialogue appears, write: No exact dialogue found.

ADAPTED SCREENPLAY SCENE:

INT./EXT. LOCATION - DAY/NIGHT

Write concise professional screenplay action in present tense.

Rules:

* If SOURCE DIALOGUE contains a spoken line, place that exact line as screenplay dialogue. Do not summarize it in action.
* Write only what can be seen or heard.
* Preserve source situational cues as short, matter-of-fact action lines.
* Convert prose into visible action, behavior, silence, sound, objects, or blocking.
* Do not explain psychology directly.
* Do not write literary narration.
* Do not invent new physical business.
* Use plain ASCII punctuation only.

Action paragraph.

CHARACTER NAME
Exact dialogue from source.

Action paragraph.

NOTES:

* What was preserved.
* What was compressed.
* What was converted from prose into visible action.
* What remains uncertain.

CONFIDENCE:
Explicit / Derived / Unclear

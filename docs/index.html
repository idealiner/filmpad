FilmPad is a screenplay editor with a built-in Local AI workspace for adapting prose source text into properly formatted screenplay scenes, powered by locally running Ollama models — no cloud, no API key.

Start in 60 seconds

  1. Download the AppImage and double-click it.
  2. If it does not open, run doctor.sh to see exactly what is missing.
  3. Copy and run the install commands shown by doctor.sh.
  4. Launch again. Writer mode works even if AI dependencies are skipped.

What’s new in v0.6

  • Script Supervisor — second-pass review agent: processes scenes one by one, compares each against adjacent context and the knowledge base, and shows a side-by-side comparison for any proposed clean-up fix (Apply & Next / Skip Scene / Stop)
  • Auto Transcript progress bar — determinate progress bar in the Writer AI panel shows how far through the document Auto Transcript (and Script Supervisor) has reached
  • Launcher Tkinter check — start-filmpad.sh now detects a missing Tkinter module before the source-run fallback and shows an actionable error dialog with the install command for the current distro

What’s new in v0.5

  • Line numbers — gutter on the left of the writer pad, stays in sync while scrolling via dlineinfo
  • Auto Transcript log — label shows Started / Block / Position / Last Saved line numbers in real time
  • Auto-save after each block — file is silently saved after every completed Auto Transcript block
  • doctor.sh — comprehensive system check: Python, Tkinter, Ollama, models, FUSE; terminal stays open after running

What’s new in v0.4

  • Launch fallback chain — launcher now tries AppImage, then dist binary, then source run for maximum reliability
  • Desktop-visible launch errors — failed starts show GUI error dialogs with recovery instructions
  • Dependency-first startup checks — Python is checked first with distro-specific install hints
  • Guided first-time setup — optional installer flow for Ollama, missing models, read-aloud tool, and spellcheck tool
  • Safe missing-Python test mode — use FILMPAD_TEST_NO_PYTHON=1 to test dependency UX without uninstalling Python

Features

  • Writer tab — screenplay editor with format presets (Scene Heading, Action, Character, Dialogue, Parenthetical, Transition), font picker, spell check, and read-aloud
  • Writer AI panel — collapsible right-side panel; select text, write a prompt or click “Transcribe into Script Format”; review output side-by-side before accepting
  • Local AI tab — split-pane workspace: load a prose source file, set a line range, pick a model, and generate scene-by-scene screenplay adaptations
  • Tab auto-save / live reload — switching tabs auto-saves and reloads the active file so both panes stay in sync
  • All AI runs locally via Ollama — no internet required after model download

Dependencies

The AppImage bundles Python and Tkinter. Only these need to be installed separately:

DependencyPurposeInstall
Ollama Local LLM inference engine (required for AI features) ollama.com/download
An Ollama model e.g. llama3, mistral, phi3 ollama pull llama3
speech-dispatcher Read-aloud (spd-say) sudo apt install speech-dispatcher

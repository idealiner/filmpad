What’s new in v0.9

  • Speech-to-text dictation at cursor — 🎤 Dictate / ⏹ Stop / × Cancel toolbar buttons; records via arecord and transcribes with whisper.cpp fully offline; text is inserted at the cursor as one undoable step; hidden automatically if tools are absent
  • Dictation Settings accordion — configurable whisper-cli executable path, GGML model file (ggml-tiny.en.bin default), language, and microphone device
  • Auto-save after every scene — Script Supervisor and Typewriter Postscript now save silently after each applied scene; log shows ✓ saved HH:MM:SS or ⚠ unsaved next to each result; sidebar status updates to Script Supervisor — scene X/Y | saved: HH:MM:SS in real time
  • Save-before-start prompt — Auto Transcript, Script Supervisor, and Typewriter Postscript prompt to save the file before starting if no file path is set, so auto-save works from the first block
  • 5-minute timeout on all model calls — Writer AI, Auto Transcript, Script Supervisor, and Typewriter Postscript all cap each Ollama call at 300 s; a stalled model is killed cleanly and the session logs timeout — skipped rather than freezing the UI indefinitely
  • Undo history cleared after each pipeline scene — edit_reset() after every Auto Transcript block and every SS/TP scene keeps RAM usage bounded during long unattended runs
  • doctor.sh Section 8 — checks arecord, whisper-cli, and GGML model files with sizes and direct download instructions

Features

  • Writer tab — screenplay editor with format presets (Scene Heading, Action, Character, Dialogue, Parenthetical, Transition), font picker, spell check, read-aloud, and speech-to-text dictation
  • Auto Transcript — block-by-block AI pass that reformats rough prose into screenplay layout; auto-saves after every block; knowledge extraction writes character and location notes to project files
  • Script Supervisor — scene-by-scene review agent with content-protection diff filter; auto-applies minor fixes (≥89% similar) and shows a side-by-side comparison for larger changes; saves after every scene
  • Script Supervisor + Style — rewrites scenes to match a style_reference.txt file while enforcing that no content is invented or removed
  • Typewriter Postscript — formatting-only final pass; converts headings, character cues, and transitions to ALL CAPS industry standard; saves after every scene
  • Local AI tab — split-pane workspace: load a prose source file, set a line range, pick a model, and generate scene-by-scene screenplay adaptations
  • All AI runs locally via Ollama — no internet required after model download

Dependencies

The AppImage bundles Python and Tkinter. Only these need to be installed separately:

DependencyPurposeInstall
Ollama Local LLM inference engine (required for AI features) ollama.com/download
An Ollama model e.g. mistral:7b, llama3.1:8b, phi3 ollama pull mistral:7b
speech-dispatcher Read-aloud (spd-say) sudo apt install speech-dispatcher
whisper.cpp + model Offline speech-to-text dictation (optional) github.com/ggerganov/whisper.cpp
alsa-utils Microphone recording for dictation (arecord) sudo apt install alsa-utils
